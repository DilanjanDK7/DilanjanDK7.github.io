fMRI Processing Pipeline for Epilepsy

Automated pipeline for preprocessing and feature extraction in epilepsy fMRI research.

← Back to Home

Overview

This repository provides a combined pipeline for processing fMRI data for epilepsy research, integrating fMRIPrep for preprocessing and a custom suite for feature extraction. Developed by Dilanjan DK at Western University.

Key Features

  • Automated DICOM to BIDS conversion (optional)
  • Preprocessing with fMRIPrep (Docker-based)
  • Feature extraction: ALFF, ReHo, Hurst, Fractal Dimension, QM-FFT, RSN
  • Flexible configuration and modular execution
  • Comprehensive documentation and troubleshooting guides
