fMRI Feature Extraction Container

Containerized workflow for advanced fMRI feature extraction and analysis.

← Back to Home

Overview

This project provides a Docker-based pipeline for extracting analytical features from resting-state fMRI data, including ALFF, fALFF, ReHo, Hurst exponent, fractal dimension, QM-FFT, and RSN analysis. The workflow uses Snakemake for management and is designed for reproducibility and scalability.

Key Features

  • ALFF, fALFF, ReHo, Hurst, Fractal Dimension, QM-FFT, RSN analysis
  • Snakemake workflow management
  • Docker containerization for reproducibility
  • Flexible configuration and modular feature selection
  • Comprehensive documentation and user guides
